Expressions are formatted in an array with:
values that represent phoneme strength: 15 digits 0<->1.0 (float)
values that represent modifier strength: 13 digits 0<->1.0 (float)
expression type: 1 digit 0<->16 (int)
expression strength: 1 digit 0<->1.0 (float)
Float values can be overset up to 3 to represent 200-300%
https://steamcommunity.com/sharedfiles/filedetails/?l=english&id=187155077 - detailed description
